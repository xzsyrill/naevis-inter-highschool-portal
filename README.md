# Naevis Inter Highschool Portal
A modern web-based school management system designed to simplify academic operations for administrators, teachers, and students.

## Live Demo
🔗 [Visit Website](https://naevisinterhigh.great-site.net/)

## Features

### 👨‍💼 Administrator
- Manage student records
- Manage teacher accounts
- Monitor school data

### 👨‍🏫 Teacher
- Manage class lists
- Input and update grades
- View student information
- Generate reports

### 👨‍🎓 Student
- View grades
- Access personal records
- Check announcements
- Monitor academic progress
